# What's new in Microsoft Entra docs for Ignite 2025
The Microsoft Entra documentation has been updated with new content and improvements to existing articles in alignment with the Ignite 2025 release. 

## Microsoft Entra ID

This release delivers major advancements in authentication, account recovery, Conditional Access Agents (including Agent ID), and privileged access management. These updates help organizations strengthen security, streamline user experiences, and simplify identity operations.

- **Authentication Methods & Passkeys**
  - Introduced and expanded documentation for passkey (FIDO2) authentication, including new guides for enabling, configuring, and managing passkey profiles.
    - [Concept: Passkeys](https://learn.microsoft.com/entra/identity/authentication/concept-authentication-passkeys)
    - [How-to: Synced Passkeys](https://learn.microsoft.com/entra/identity/authentication/how-to-authentication-synced-passkeys)
    - [Persona-based Passwordless Planning](https://learn.microsoft.com/entra/identity/authentication/how-to-plan-persona-phishing-resistant-passwordless-authentication)
    - [Authentication Methods Overview](https://learn.microsoft.com/entra/identity/authentication/concept-authentication-methods)
  - Expanded and clarified instructions for certificate-based authentication, voice call authentication, and MFA app passwords.

- **Account Recovery**
  - New and revised documentation for Microsoft Entra account recovery and self-service account recovery (SSAR), including setup guides, troubleshooting, and workflow overviews.
    - [How-to: Account Recovery User Setup](https://learn.microsoft.com/entra/identity/authentication/how-to-account-recovery-user-setup)
    - [Concept: Self-Service Account Recovery](https://learn.microsoft.com/entra/identity/authentication/concept-self-service-account-recovery)
  - Expanded FAQs and clarified geographic scope and recovery phases.
  - Added new screenshots and preview steps to improve user and admin experience.

- **Privileged Identity Management (PIM) & Access Packages**
  - General Availability (GA) of Dynamic Approvals for access packages.
    - [How-to: Add Assignments](https://learn.microsoft.com/entra/identity/privileged-identity-management/pim-how-to-add-assignments)
  - New documentation for assigning eligible access packages and updates to PIM API statuses and descriptions.
    - [PIM API Overview](https://learn.microsoft.com/entra/identity/privileged-identity-management/pim-api-overview)

- **Conditional Access & Agent ID**
  - Major updates to Conditional Access Agents, including the introduction of Agent ID platform roles and RBAC (role-based access control).
    - [Conditional Access Overview](https://learn.microsoft.com/entra/identity/conditional-access/)
  - Added core Agent ID documentation, batch agent ID drafts, and expanded agent management capabilities.
  - Multiple releases and documentation updates for Conditional Access Agents, supporting broader deployment and automation.

- **Device Join & Health Agents**
  - Updated guides for device join via Microsoft Entra company portal.
    - [How-to: Health Agent Install](https://learn.microsoft.com/entra/identity/hybrid/connect/how-to-connect-health-agent-install)
  - Revised installation requirements and troubleshooting for health agents.

## Microsoft Entra Permissions Management

This release includes new content and updates to permissions management, focusing on access package assignment and dynamic approvals to help organizations manage privileged access more securely and efficiently.

- General Availability of Dynamic Approvals for access packages.
  - [How-to: Add Assignments](https://learn.microsoft.com/entra/identity/privileged-identity-management/pim-how-to-add-assignments)
- New and updated guides for assigning eligible access packages in PIM.
- Updates to API statuses and descriptions for PIM features.
  - [PIM API Overview](https://learn.microsoft.com/entra/identity/privileged-identity-management/pim-api-overview)

## Microsoft Entra External ID

Enhancements to external identity documentation help organizations manage customer and partner identities more effectively.

- New and updated FAQs for external identities and customer scenarios.
  - [FAQ: External ID Customers](https://learn.microsoft.com/entra/external-id/customers/faq-customers)
- Improved guidance for configuring and managing external user access.

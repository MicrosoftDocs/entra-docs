# Conditional Access and agent identities (Preview)

There are three main scenarios covered by the preview

- Stop all agent use within your organization
    -  All Agents, All Resources, Block.
- Allow specific testing of agents against organization resources
    - All Agents - excluding specific test agent, All Resources, Block
- Allow specific agents to be accessed by users as a resource
    - All users, All agents except specific test agents, Block
